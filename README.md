# 🐾 Lar Patinhas Felizes

App mobile de adoção de pets feito com React Native + Expo.

A ideia é simples: uma tela de login e, depois de entrar, você navega pelo lar e vê os pets disponíveis para adoção.

---

## O que tem no app

- **Tela de login** com validação de usuário e senha
- **Cabeçalho** com foto e nome do lar
- **Banner** de chamada para adoção
- **Carrossel** de pets disponíveis com foto, raça e status
- **Sobre nós** com endereço, horário e contato
- **Perfil** do usuário logado

---

## Como rodar

**Pré-requisitos:** Node.js e o app Expo Go instalado no celular

```bash
# Clone o repositório
git clone https://github.com/SEU_USUARIO/NOME_DO_REPO.git

# Entre na pasta
cd NOME_DO_REPO

# Instale as dependências
npm install

# Rode o projeto
npx expo start
```

Depois é só escanear o QR code com o **Expo Go** no celular.
